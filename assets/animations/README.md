# Animation Assets for StuDex

This directory contains Lottie animation files for enhanced user experience.

## Required Animation Files:
- `loading.json` - Loading animation for async operations
- `success.json` - Success animation for completed actions
- `error.json` - Error animation for failed operations
- `study-session.json` - Animation for study session start
- `quiz-complete.json` - Animation for quiz completion

## Usage:
These animations are used throughout the app to provide visual feedback and enhance the user experience during various operations.

## File Format:
- Format: Lottie JSON
- Size: Optimized for web and mobile
- Duration: 2-5 seconds
- Loop: Single play for success/error, loop for loading
