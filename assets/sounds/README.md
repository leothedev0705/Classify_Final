# Sound Assets for StuDex

This directory contains ambient sound files for the study session feature.

## Required Sound Files:
- `rain.mp3` - Ambient rain sounds for focus
- `cafe.mp3` - Cafe ambience for study sessions  
- `nature.mp3` - Nature sounds for relaxation
- `white-noise.mp3` - White noise for concentration

## Usage:
These sounds are used in the Study Session feature when users select ambient sounds for their study sessions. The sounds help create a focused environment for productive studying.

## File Format:
- Format: MP3
- Quality: 128kbps or higher
- Duration: 10-30 minutes (looped)
- Volume: Normalized to prevent jarring volume changes
